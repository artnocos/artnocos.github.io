POSTを書き始めようとしたところ
Markdownは改行を入れるのが面倒なので
先ずはTextileで書けるようにした（LLMができると言うので教えてもらった）
もっと調整が必要かもしれないがとりあえずこのぐらいで
_（ちなみに Markdown で改行させるのはバックスラッシュを使うのが一番マシだと思う）_

プラグイン jekyll-textile-converter を追加までは難しくなかったが
コードブロックはいろいろやらないとそれっぽくならなかった
空行なしで前の文章に続けてブロックを書くと間隔が大きくなりすぎるようだ

{% highlight shell %}
export ABC=123 # shell
{% endhighlight %}
{% highlight c %}
int a = 3; // C
{% endhighlight %}
{% highlight ruby %}
r = nil # Ruby
{% endhighlight %}
{% highlight python %}
odds = [i for i in range(10) if i % 2 == 1] #Python
{% endhighlight %}
後は続けて書いても大丈夫

bc(c). int h = 0; //bcだとハイライトなし
